f99투자하기 좋은 종목 선별 컬럼

2주대비 오늘 거래비율		: 70% ~ 130%
투자의견			: 매수 or 적극매수
현재상황 대비 목표주가		: 50% 이상
최근 52주 대비 현재 위치 	: 무릎 or 발

=============================================

EPS(주당순이익) = 당기순이익 / 유통주식수
-> EPS는 높을수록 좋다. (배당여력이 크다는 의미이기 때문) (기업 가치도 높고 안정적인 기업이라 여길 수 있다.) (투자기준을 잡을수도 있을 정도로 중요한 정보)

PER(주가수익비율) = 주가 / EPS
-> 일반적으로 낮을수록 저평가, 높으면 고평가. (예외, 개별 종목의 경우 성장주는 PER이 투자심리를 반영해 훨씬 높은경우가 다반수임, 회사 자체에 문제가 있어서 저 PER인 경우도 많음)

BPS(주당 순자산 가지) = 순자산(자산 - 부채) / 유통주식수
-> BPS가 높을수록 좋다. (주가와 상관관계는 별로 없음)

PBR(주가 순자산비율) = 주가 / BPS
-> 일반적으로 낮을수록 높은 주가 수익률을 나타냄 (회사 가치평가를 위해 많이 사용됨, PER과 함께 많이 사용), (경영성과 좋을수록 PBR배율이 높게 나타남)

ROE(자기 자본 이익률) = 당기순이익/자기자본 * 100
-> 낮을수록 경영진이 무능하거나 해당 업종이 불황인 case가 많음

* 추가적인 공식 => PBR = PER * ROE
